Запуск приложения
1 Скопировать с githab https://github.com/slivlikh/react-test
2 Импортировать БД webby-lab.sql MySQL, движок - InnoDB
3 В файе config.json указать: хост, имя базы, логин, пароль, порт и порт который будет слушать нода
4 Запустить из каталога с приложениям node app.js  
Архитектура
1 Front end
Одностриничное приложение написанное с помощью библиотеки ReactJS
Родительский компонент ReactApp
Компонент AddNewFilm отвечает за создание и отправку на сервер нового фильма
Компонент SendFile  отвечает за прием и отпрвку на сервер файла и импортом фильмов
Компонент SearchForm отвечает за поиск фильмов по актеру и поиск фильма
Компонент FilmsList отвечает за вывод текущего списка фильмов
Компонент Film отвечает за вывод одного фильма, а также отправляет запрос на сервер за доп. информацией о фильме и через компонент FilmsList к ReactApp эмиртирует событие удаления фильма, также отправляет запрос на сервер для удаления фильма из БД


Иерархия компонентов

ReactApp
	AddNewFilm
	SendFile
	FilmsList
		Film
	SearchForm
2 Back end
Написан на nodeJS с помощью фреймворка express и БД MySQL
Back end предоставляет restfull api для:
	- создания фильма (метод PUT)
	- импорта фильмов из файла (метод PUT)
	- отображения списка всех фильмов (метод GET)
	- отображения подробный информации о фильме ( метод GET)
	- поиск фильма по названию и по актеру (метод GET)
	- удаления фильма (метод DELETE)
